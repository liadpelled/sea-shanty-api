Sea Shanty API

Add some random sea wisdom to your project.

Ahoy, matey! What have ye here?

This API generates a random excerpt from a large collection of sea shanties.

How to Use

Get a random excerpt from a sea shanty along with its title:

https://sea-shanty-api.herokuapp.com/random

Get a list of all shanties:

https://sea-shanty-api.herokuapp.com/shanties/all

Get the lyrics of a shanty:

https://sea-shanty-api.herokuapp.com/shanties?title={TITLE}

The ocean is dark and full of errors!

If you spot an error (or have a suggestion) feel free to message me on Reddit!
